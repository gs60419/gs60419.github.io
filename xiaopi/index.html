從差異到六利的世界語法

小屁論

一套從「差異」出發，推進到人類心智、行為、語意與 AI 語法生命的理論體系。它的任務，是讓每個概念都能回溯到差異，並沿生成鏈找到自己的位置。

看核心鏈 GitHub

不是玩笑

「小屁」是差異被觀測後外翻出來的最小可見表面；小屁論研究它如何被觀測、投影、反投影並生成意義。

差異第一因

沒有差異，就沒有相對；沒有相對，就沒有投影、判斷、交換與生成。差異是世界能被分辨和運算的最小條件。

六利座標

人不是只做單一選擇，而是在生理、生存、社會、認知、敘事與意義之間，最大化某種利的組合。

Core Chain

核心生成鏈

  1. 差異任兩個狀態之間可被分辨的條件。
  2. 偏壓差異形成方向、權重與限制。
  3. 三光譜差異光譜、存在光譜、語法光譜。
  4. 六利六個行為驅動向度。
  5. 交換率投入、代價、回報與主觀划算程度。
  6. 行為 / 投影概念進入選擇、行動與回饋。

Six Interests

六利系統

Δ生理利

身體與能量是否划算。

π生存利

環境與風險是否可承受。

ψ社會利

關係與位置是否成立。

φ認知利

理解成本是否降低。

η敘事利

時間與故事是否連續。

Ω意義利

價值與方向是否值得。
